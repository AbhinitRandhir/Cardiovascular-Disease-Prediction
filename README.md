# ❤️ Cardiovascular Disease Prediction Using Machine Learning

A Machine Learning-based web application developed using **Python, Flask, Scikit-Learn, HTML, CSS, and JavaScript** to predict the risk of cardiovascular disease based on patient health parameters. This project provides an intuitive web interface where users can enter medical details and receive an instant prediction.

---

## 📌 Project Overview

Cardiovascular disease is one of the leading causes of death worldwide. Early prediction can help doctors and patients take preventive measures before serious complications occur.

This application uses a trained **Random Forest Machine Learning model** to analyze patient health data and predict the likelihood of cardiovascular disease. The prediction is displayed through a simple, responsive, and user-friendly web interface.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/AbhinitRandhir/Cardiovascular-Disease-Prediction.git
```

### Navigate to the Project

```bash
cd Cardiovascular-Disease-Prediction
```

### Create a Virtual Environment

```bash
python -m venv venv
```

### Activate the Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux / macOS**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
